# Animation Improvements Plan

## Goal
Improve portfolio animations to be more engaging while keeping a minimal & subtle style.

## Enhancements to Implement

### 1. Enhanced Parallax
- Keep existing parallax effect but reduce intensity for subtlety
- Add subtle vertical offset to sections on scroll
- Smooth out the motion with better easing

### 2. Entrance Animations
- Add fade-in + slide-up animations when sections enter viewport
- Use IntersectionObserver (already partially implemented in useScrollAnimation.js)
- Apply to all major sections: Hero, Portfolio, Skills, Testimonials, Contact
- Duration: ~600ms with ease-out timing

### 3. Staggered Lists
- Animate project cards, skill bars, and testimonials with delay between each item
- Stagger delay: ~100ms between items
- Trigger on scroll into view

### 4. Animated Background
- Replace static background image with subtle animated gradient/mesh
- Add gentle floating shapes or particles (low opacity, slow movement)
- Keep transitions smooth when section colors change

### 5. Micro-interactions
- Button hover: subtle scale + background color transition
- Card hover: gentle lift (translateY) + shadow increase
- Skill bars: smooth fill animation when revealed
- Links: underline slide animation

## Implementation Steps

### Step 1: Update globals.css

### Step 4: Update components with entrance animations
- HeroSection: fade in on load

### Step 5: Add micro-interactions
